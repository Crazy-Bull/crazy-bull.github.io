Codeforces 1838E Count Supersequences

Posted on 2023-08-27 In OI

题意

    有一个长为 n 的数组 a，元素均为 [1, k] 内的整数，求有多少个长为 m 的数组 b，元素同样为 [1, k] 内的整数，
且包含 a 为其子序列，答案对 1e9+7 取模。

* 注：子序列可以不连续

数据范围

1 <= n <= 2·105
n <= m <= 109
1 <= k <= 109
1 <= ai <= 109

题解

数据范围要求时间复杂度应尽可能避免与 m，k 有关，因此朴素的 dp 是行不通的。但是我们不妨先考虑如果 n, m 不大时的 dp 做法。事实上，这会导出一个有用的结论，从而把时间复杂度优化到 O(nlogM) （此处 M 指模数）

令 dp[i][j] 为包含前 i 个元素为其子序列，长为 j 的数组个数。则不难得出状态转移方程：


初始条件由 dp 数组的意义显然，转移方程的第一项代表 bj 不为 ai 时，b 的前 j-1 项包含 a 的前 i 项为子序列，此时 bj 共有 k-1 种取值；同理，第二项代表 bj 为 ai 时，b 的前 j-1 项包含 a 的前 i-1 项为子序列，，此时 bj 只有 1 种取值（即ai）

我们注意到，状态转移方程与数组 a 完全无关，这也意味着答案与 ai 的取值无关，从而我们可以设 ai = 1 (1 <= i <= n)
