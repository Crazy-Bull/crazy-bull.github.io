Codeforces 1862G The Great Equalizer

Posted on 2023-08-29 In OI

题意

（题目原表述比较复杂，这里描述得尽可能简单一些）
对于长为 n 的数列 a，定义一次操作如下几步构成：

1. 将 a 按升序排列，并去除重复元素
2. 若 a 只剩一个元素，结束，否则转向 3
3. 对 1 <= i <= n，将 a[i] 加上 (n-i+1)
对 a 进行若干次（可能为 0 次）操作后，a 最后只剩下一个数，记为 f(a)，注意 n 在两次操作间可能会变化。
现给定一个长为 n 的初始数列 a 和 q 次询问，每次询问给定两个数 i,x，并重新赋值 a[i] = x，
（每次赋值在后续询问中保留），求每次询问后的 f(a)

数据范围

1 <= n, q <= 2·105
1 <= ai, x <= 109
1 <= i <= n

Codeforces 1838E Count Supersequences

Posted on 2023-08-27 In OI

题意

    有一个长为 n 的数组 a，元素均为 [1, k] 内的整数，求有多少个长为 m 的数组 b，元素同样为 [1, k] 内的整数，
且包含 a 为其子序列，答案对 1e9+7 取模。

* 注：子序列可以不连续

数据范围

1 <= n <= 2·105
n <= m <= 109
1 <= k <= 109
1 <= ai <= 109

Codeforces 1859D Andrey and Escape from Capygrad

Posted on 2023-08-20 Edited on 2023-08-25 In OI

题意

    有n个传送门，每个传送门有四个参数 l <= a <= b <= r，表示当你的位置在 [l, r] 上时，可以选择传送到 [a, b] 上的任意一个点。
    随后有 q 个询问，每个询问给出一个坐标 x 代表你的起始位置，对于每个输入 x，输出通过传送门能最右到达的位置。

数据范围

1 <= n <= 2⋅105
1 <= l <= a <= b <= r <= 109
1 <= q <= 2⋅105
1 <= q <= 109
